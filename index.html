Automations & Systems Built

Legal Case Documentation Pipeline
Two-phase Python automation (prepare.py + merge.py) that downloads discounting agreements from Bubble.io CDN, extracts investor names via text parsing, organizes Drive folders per investor, and assembles complete legal PDF packages — agreement pages, signature page, DigiO audit trail, and back pages — ready for legal submission.
↓ 2–3 days of manual work → minutes per asset · 100–150 investors per case
Python Google Drive API Bubble.io CDN PDF Parsing

TDS Reconciler
Bulk-extracts PAN numbers and TDS values from 2,000+ Form 16A PDFs and matches them against internal TDS master records — generating structured match/mismatch/variance reports.
↓ ~90–95% reduction in reconciliation time
Python Excel

Agreement Alignment Checker
Python tool using coordinate-based checking to verify signing page placement across Discounting Agreements, Guarantee Agreements, Indemnity Bonds, and Term Sheets before execution — results in ~2 seconds.
Eliminated repeated manual rechecks pre-execution
